Usage

  • API domain: https://api.itooi.cn/
  • API Options:
    • song
    • url
    • lrc
    • pic
    • songList
    • and so on…
  • API format: API domain + /tencent + API Options + Params

General parameters

isRedirect

Whether to redirect to the obtained http address.

At present, all music interfaces can use this parameter to obtain the song playback address, picture address, and Mv playback address.

The specific description of each interface shall prevail

  • value:
    • 0 for disable
    • 1 for enable

format

Whether to format the JSON.

  • value:
    • 0 for disable (default)
    • 1 for enable (recommended)

Options details

song

Return the information of songs.

id

Music ID (Which you can found in music website’s url).

Separate with commas.

url

Resouce addr.

quality

  • value:
    • 42
    • 96
    • 128
    • 192
    • 320
    • ape
    • flac

The value bigger, the Sound quality better.

lrc

Lyrics with timeline.

pic

Get the poster of songs.

imgsize

Default 200x200.

songList

Get all the music of playlists.

Example

https://api.itooi.cn/tencent/songList?id=1156824473&&format=1

Done.
